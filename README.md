# Discord-Ticket-Claimer
Requires token - writes custom message when channel is created that has keyword inside. Use a settings.json file e.g 
'''{
    "settings": [{
        "token": "Token",
        "delay":1,
        "message":"First",
        "serverid":1212121211,
        "keyword":"ticket",
        "key":1
    }
    ]
}'''
